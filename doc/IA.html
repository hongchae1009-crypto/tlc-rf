IA — TLC Rf 계산기

Information Architecture · v1.0 · 2026-06-19

1. 구조 개요

단일 페이지(SPA) · 5단계 마법사 + 모달. 라우팅 없음. 상태(S.step)로 화면 전환.

TLC Rf 계산기 (단일 페이지)
├─ 상단바(Topbar)    — 로고 · ⚡전자동 분석 · 새 이미지 · ❓도움말
├─ 스텝퍼(Stepper)   — ① 불러오기 → ② 보정 → ③ 기준선 → ④ Spot → ⑤ 결과
├─ 본문(Main)
│   ├─ 무대(Stage)   — 드롭존 또는 캔버스(오버레이) · [모바일] 줌 컨트롤
│   └─ 패널(Panel)   — 단계별 안내·컨트롤 + 하단 이전/다음
└─ 모달/오버레이      — 도움말 · 토스트

2. 네비게이션

  • 선형 진행: 패널 하단 이전 / 다음. 스텝퍼 라벨 클릭 이동(보정 후 3~5단계 접근).
  • 점프: ⚡ 전자동 → 일괄 처리 후 ⑤ 결과로 이동.
  • 리셋: 새 이미지 → ①로 초기화.

3. 화면별 구성

단계무대(Stage)패널(Panel) 컨트롤
① 불러오기드롭존(클릭/드래그/Ctrl+V)입력 방식 안내
② 보정캔버스 + 코너 4핸들 + 돋보기회전 ↺↻ · 플레이트 자동 인식 · 전체(보정 생략)
③ 기준선캔버스 + baseline·용매front 선분자동 제안 · 평행 이동 슬라이더
④ Spot캔버스 + spot 마커·레인 핸들감도 · 다시 검출 · 선택/모두 삭제 · 레인 ±
⑤ 결과캔버스 + 마커·Rf 라벨(겹침 회피)Rf 표 · RF 글씨 크기 · 내보내기 4종

5. 상태 전이

  • goStep(n)이 단계 전환. ②→③ 시 buildRect()(워프)·autoLines(), ③→④ 시 미검출이면 detectSpots().
  • 뷰 이미지: ②=작업(work), ③~⑤=보정(rect).
  • ⑤에서 캔버스 0.5초 길게 누름 → 마커 없는 원본 미리보기(떼면 복귀).

6. 컴포넌트 인벤토리

  • 입력: 파일 input(hidden), 드래그앤드롭, paste.
  • 오버레이 핸들: 코너(4), 기준선 끝점(좌/우×2선), spot 마커, 레인 핸들, 삭제 ✕ 배지(모바일), 돋보기.
  • 컨트롤: 버튼, 슬라이더(감도·라벨크기·평행이동), 줌(±/fit), 토스트.
  • 모달: 도움말.

7. 반응형

  • ≥681px: 좌(무대) + 우(패널 300~340px) 2열.
  • ≤680px: 1열 세로 스택. 캔버스 고정 크기 + transform 줌/팬, 이전/다음 하단 고정.
