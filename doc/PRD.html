PRD — TLC Rf 계산기

Product Requirements Document · v1.0 · 2026-06-19 · 라이브 · 저장소

1. 개요

TLC(박층 크로마토그래피) 플레이트 사진을 업로드하면 baseline(원점)·전개용매선·각 spot 위치를 잡아 Rf 값을 자동 계산해 주는 단일 파일 웹 도구. 순수 JavaScript로 작성되어 외부 의존성 없이 완전 오프라인으로 동작한다(index.html 하나).

2. 배경 / 문제

  • TLC 결과를 손으로 자를 대어 Rf를 계산하는 일이 번거롭고 재현성이 낮다.
  • 사진이 카메라 각도로 기울어지거나(평행사변형) 배경이 함께 찍힌다.
  • UV254 형광, 발색(stain) 등 플레이트 종류가 다양해 일괄 처리가 어렵다.

3. 목표 / 비목표

목표

  • 사진 1장 → Rf 표를 5단계 마법사로 빠르게 산출.
  • 기울어진 사진 원근 보정, spot 자동 검출(+수동 보정), 결과 내보내기.
  • 설치 없이 PC·모바일에서 사용. 사진은 기기 밖으로 나가지 않음.

비목표

  • 계정/서버/DB, 영구 저장, 다중 사용자 협업. 정량(농도) 분석.

4. 사용자

1차: 유기/천연물 화학 연구자·대학원생. 사용 맥락: 실험 직후 TLC 결과를 ELN/보고서에 기록.

6. 비기능 요구사항

  • 오프라인·단일 파일: CDN/외부 폰트·스크립트 없음. 더블클릭 동작.
  • 프라이버시: 이미지 전송 없음(브라우저 내 처리).
  • 성능: 워프 ≤1000px, 작업 이미지 ≤1200px 다운스케일.
  • 호환성: 최신 Chrome/Safari/Edge(PC·모바일). Pointer Events·Canvas2D.

7. 성공 기준

  • 기울어진 실제 UV/stain 사진의 코너 보정 후 직사각형화 성공.
  • 자동 검출이 spot 다수를 잡고, 감도/수동으로 100% 교정 가능.
  • 같은 화합물이 여러 레인에서 유사 Rf(±0.03).
  • 내보내기가 기울어진 선분·spot 좌표 기준으로 정확.

8. 기술 스택

  • HTML5 + Canvas2D + 바닐라 JS(ES2020), 빌드 없음.
  • 영상처리 자체 구현: 호모그래피(8×8 가우스 소거 + bilinear), 박스 블러, 연결요소 blob, 중앙값 배경 추정, 1D 레인 클러스터링.
  • 배포: GitHub Pages(main 루트), git push 자동 재배포.

9. 로드맵

  • 보정 전 원본 사진 내보내기 옵션, 멀티 플레이트 일괄 처리.
  • 검출 임계 프리셋(UV/stain), 레퍼런스 Rf 비교, PDF/ELN 연동.
