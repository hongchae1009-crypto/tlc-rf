# PRD — TLC Rf 계산기

> Product Requirements Document · v1.0 · 2026-06-19
> 라이브: https://hongchae1009-crypto.github.io/tlc-rf/ · 저장소: https://github.com/hongchae1009-crypto/tlc-rf

## 1. 개요
TLC(박층 크로마토그래피) 플레이트 사진을 업로드하면 baseline(원점)·전개용매선·각 spot 위치를 잡아
**Rf 값을 자동으로 계산**해 주는 단일 파일 웹 도구. 순수 JavaScript로 작성되어 외부 의존성 없이
**완전 오프라인**으로 동작한다(`index.html` 하나).

## 2. 배경 / 문제
- 연구실에서 TLC 결과를 손으로 자를 대어 Rf를 계산하는 일이 번거롭고 재현성이 낮다.
- 사진은 카메라 각도 때문에 기울어지거나(평행사변형) 배경이 함께 찍히는 경우가 많다.
- UV254 형광 플레이트, 발색(stain) 플레이트 등 종류가 다양해 일괄 처리가 어렵다.

## 3. 목표 / 비목표
**목표**
- 사진 1장 → Rf 표를 5단계 마법사로 빠르게 산출.
- 기울어진 사진 원근 보정, spot 자동 검출(+수동 보정), 결과 내보내기.
- 설치 없이 PC·모바일 브라우저에서 바로 사용. 사진은 기기 밖으로 나가지 않음(프라이버시).

**비목표**
- 계정/서버/DB, 데이터 영구 저장, 다중 사용자 협업.
- 정량 분석(농도 정량), 스펙트럼 분석.

## 4. 사용자
- 1차: 유기/천연물 화학 연구자·대학원생(NMR·MS·합성·바이오어세이 병행).
- 사용 맥락: 실험 직후 TLC 결과를 ELN/보고서에 기록.

## 5. 기능 요구사항

| ID | 기능 | 설명 |
|----|------|------|
| F1 | 이미지 불러오기 | 파일 선택 · 드래그&드롭 · 클립보드 붙여넣기(Ctrl+V). 원본/잘린 이미지 모두 |
| F2 | 플레이트 원근 보정 | 네 모서리 드래그 → 호모그래피 워프로 직사각형화. 코너 드래그 시 돋보기 |
| F3 | 플레이트 자동 인식 | 형광 녹색/배경 대비/밝기 기반으로 플레이트 영역 자동 코너 추정(색 무관) |
| F4 | 이미지 회전 | 가로 사진 90°(↺/↻) 회전 |
| F5 | 기준선 지정 | baseline·용매 front 선분. 가운데=평행 이동, 양 끝=기울기 조정(보간 Rf) |
| F6 | Spot 자동 검출 | 배경색 대비 2D blob 검출 + 감도 슬라이더. 레인 자동 클러스터링 |
| F7 | Spot 수동 편집 | 빈 곳 탭=추가, 드래그=이동, 삭제(PC 우클릭·Delete / 모바일 탭→✕) |
| F8 | 레인 관리 | 레인 핸들 이동·추가·삭제, 레인 라벨(화합물명) 입력 |
| F9 | Rf 계산·결과표 | Rf=(원점→spot)/(원점→용매front). 레인·spot별 표. RF 라벨 글씨 크기 조절 |
| F10 | 내보내기 | 표 복사(TSV) · CSV · 주석 이미지(PNG) · 원본 이미지(PNG) |
| F11 | 전자동 모드 | ⚡ 전 과정 일괄 처리 후 결과로 이동(검토·수정 가능) |
| F12 | 되돌리기 | Ctrl+Z (코너·기준선·spot·레인 편집) |
| F13 | 모바일 대응 | 세로 스택 레이아웃, 줌(버튼/핀치/두 손가락 팬), 터치 편집, 길게눌러 원본 미리보기 |
| F14 | 도움말 | 상단 ❓ 도움말 모달, 첫 방문 1회 자동 표시 |

## 6. 비기능 요구사항
- **오프라인·단일 파일**: CDN/외부 폰트/스크립트 없음. `index.html` 더블클릭으로 동작.
- **프라이버시**: 이미지 업로드 전송 없음(브라우저 내 처리).
- **성능**: 워프 ≤1000px, 작업 이미지 ≤1200px 다운스케일로 즉시 반응.
- **호환성**: 최신 Chrome/Safari/Edge(PC·모바일). Pointer Events·Canvas2D 사용.
- **유지보수**: 단일 HTML, 디자인 토큰 재사용(lab-kiosk/E-Note ELN 계열).

## 7. 성공 기준
- 기울어진 실제 UV/stain 사진에서 코너 보정 후 직사각형화 성공.
- 자동 검출이 실제 spot 다수를 잡고, 감도/수동 보정으로 100% 교정 가능.
- 같은 화합물이 여러 레인에서 유사 Rf(±0.03) 산출.
- 표 복사/CSV/PNG 내보내기가 기울어진 선분·spot 좌표 기준으로 정확.

## 8. 기술 스택
- HTML5 + Canvas2D + 바닐라 JS(ES2020). 빌드 없음.
- 영상처리 자체 구현: 호모그래피(8×8 가우스 소거 + bilinear 워프), 박스 블러,
  연결요소(flood fill) blob 검출, 중앙값 배경 추정, 1D 레인 클러스터링.
- 배포: GitHub Pages(main 루트). `git push` → 자동 재배포.

## 9. 로드맵(향후)
- 원본(보정 전) 사진 내보내기 옵션, 멀티 플레이트 일괄 처리.
- 검출 임계 프리셋(UV/stain), Rf 비교(레퍼런스 라인).
- 결과 PDF/ELN 연동.
