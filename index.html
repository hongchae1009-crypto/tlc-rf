TLC Rf 계산기 사용법

TLC 사진을 올리면 5단계로 Rf 값을 계산합니다. 완전 오프라인 · 사진은 기기 밖으로 나가지 않습니다.

1불러오기

  • 사진 파일 선택 · 화면으로 드래그&드롭 · 클립보드는 Ctrl+V 붙여넣기
  • 원본 사진(배경 포함)도, 플레이트만 잘라낸 이미지도 모두 가능

2플레이트 보정

  • 파란 점 4개(모서리)를 플레이트 네 귀퉁이에 맞춰 드래그 → 기울어진 사진도 직사각형으로 펴짐
  • 드래그할 때 돋보기로 모서리가 확대되어 정밀 조정 가능
  • 플레이트 자동 인식 버튼으로 코너 자동 잡기 · ↺ ↻로 가로 사진 90° 회전
  • 이미 직사각형으로 잘린 이미지면 전체(보정 생략)

3기준선

  • baseline(원점)과 용매 front 선을 맞춤
  • 선 가운데를 잡으면 평행 이동, 양 끝을 잡으면 기울기 조정(기울어진 플레이트 대응)

4Spot 확인

  • 자동 검출된 spot 확인 · 못 잡으면 감도 슬라이더를 올림
  • 빈 곳 탭/클릭=추가, 드래그=이동
  • 삭제 — PC: 우클릭 또는 선택 후 Delete / 모바일: 탭해 선택 → 빨간 ✕
  • 레인(시료 열)은 자동 묶임 · 레인 번호 핸들 드래그로 위치 조정, 우클릭/✕로 레인 삭제

5결과 & 내보내기

  • 레인·spot별 Rf 표 · 레인 라벨(화합물명) 입력 가능 · RF 글씨 크기 슬라이더
  • 내보내기 — 복사(Ctrl+V로 바로 붙여넣기): 표+이미지 한 번에 · 표만 · 주석 이미지 · 원본 이미지
  • 내보내기 — 파일 저장: CSV · 주석 이미지 · 원본 이미지 · 스팟+원본 1장
  • 이미지를 0.5초 길게 누르면 마커 없는 원본이 보임(떼면 복귀)
💡 ⚡ 전자동 분석은 전 과정을 한 번에 처리한 뒤 결과로 이동합니다(④에서 검토·수정 가능). · Ctrl+Z 되돌리기 · 모바일은 ＋ / ⤢ / − 버튼·핀치·두 손가락으로 확대/이동.
Rf = (원점→spot 거리) ÷ (원점→용매front 거리), 0~1.
