# TLC Rf 계산기

TLC(박층 크로마토그래피) 플레이트 사진을 올리면 Rf 값을 자동으로 계산해 주는 단일 파일 웹 도구입니다.
순수 JavaScript로 작성되어 **외부 의존성 없이 완전 오프라인**으로 동작합니다 (`index.html` 파일 하나).

## 사용 흐름

1. **불러오기** — 사진 파일 선택 / 드래그&드롭 / 클립보드 붙여넣기(Ctrl+V). 원본 사진·잘라낸 이미지 모두 가능.
2. **플레이트 보정** — 네 모서리를 드래그하면 기울어진 사진도 직사각형으로 펴집니다(원근 보정). 플레이트 자동 인식 지원.
3. **기준선** — baseline(원점)·용매 front 선을 지정. 가운데를 잡으면 평행 이동, 양 끝을 잡으면 기울기 조정.
4. **Spot** — spot을 자동 검출(배경색 대비). 드래그 이동·탭으로 추가·개별 삭제, 감도 슬라이더로 민감도 조절.
5. **결과** — 레인·spot별 Rf 표 + 클립보드 복사 / CSV / 주석 PNG 내보내기.

`⚡ 전자동` 버튼으로 전 과정을 한 번에 처리한 뒤 검토할 수도 있습니다. 모바일(세로) 레이아웃과 터치 편집(탭→✕ 삭제)도 지원합니다.

## 실행

- 브라우저에서 `index.html`을 더블클릭하면 바로 사용할 수 있습니다(오프라인).
- 또는 GitHub Pages에 배포된 주소로 접속하세요.

Rf = (원점 → spot 거리) / (원점 → 용매 front 거리), spot의 x 위치에서 기준선을 보간해 계산합니다.
